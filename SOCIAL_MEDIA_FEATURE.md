# إضافة روابط التواصل الاجتماعي

## الميزات المضافة

تم إضافة نظام كامل لإدارة روابط التواصل الاجتماعي في المتجر مع الميزات التالية:

### 1. قاعدة البيانات (Backend)

#### Schema جديد: `StoreSettings`
- **الموقع**: `server/schemas-extended.ts`
- **الحقول**:
  - `storeName`: اسم المتجر (اختياري)
  - `storeDescription`: وصف المتجر (اختياري)
  - `storeLogo`: شعار المتجر (اختياري)
  - `socialMedia`: كائن يحتوي على روابط التواصل الاجتماعي:
    - `facebook`
    - `twitter`
    - `instagram`
    - `tiktok`
    - `youtube`
    - `whatsapp`
    - `linkedin`

#### API Endpoints
- **الموقع**: `server/routers-extended.ts`
- **Endpoints**:
  - `storeSettings.get`: للحصول على إعدادات المتجر (عام - يمكن للجميع الوصول إليه)
  - `storeSettings.update`: لتحديث إعدادات المتجر (محمي - للمسؤولين فقط)

#### دوال قاعدة البيانات
- **الموقع**: `server/db-mongo-extended.ts`
- **الدوال**:
  - `getStoreSettings()`: للحصول على الإعدادات
  - `updateStoreSettings(data)`: لتحديث الإعدادات

### 2. لوحة التحكم (Admin Panel)

#### صفحة إعدادات المتجر
- **الموقع**: `client/src/pages/admin/StoreSettings.tsx`
- **المسار**: `/admin/store-settings`
- **الميزات**:
  - نموذج لتعديل معلومات المتجر الأساسية
  - حقول منفصلة لكل منصة تواصل اجتماعي
  - أيقونات ملونة لكل منصة
  - حفظ تلقائي للإعدادات
  - رسائل نجاح/فشل

#### التكامل مع صفحة الإعدادات الرئيسية
- **الموقع**: `client/src/pages/admin/AdminSettings.tsx`
- تم إضافة رابط إلى صفحة إعدادات المتجر من بطاقة "المتجر"

### 3. واجهة المستخدم (Frontend)

#### مكون روابط التواصل الاجتماعي
- **الموقع**: `client/src/components/SocialMediaLinks.tsx`
- **الميزات**:
  - عرض أيقونات التواصل الاجتماعي فقط إذا كانت الروابط موجودة
  - تصميم أنيق مع تأثيرات hover
  - أيقونات مخصصة لـ TikTok و WhatsApp
  - روابط تفتح في نافذة جديدة

#### التكامل مع الصفحة الرئيسية
- **الموقع**: `client/src/pages/Home.tsx`
- تم إضافة قسم لروابط التواصل الاجتماعي في الـ Footer

### 4. التوجيه (Routing)
- **الموقع**: `client/src/App.tsx`
- تم إضافة مسار جديد: `/admin/store-settings`

## كيفية الاستخدام

### للمسؤول:

1. قم بتسجيل الدخول إلى لوحة التحكم
2. انتقل إلى "الإعدادات" → "المتجر"
3. أدخل روابط التواصل الاجتماعي الخاصة بك
4. احفظ الإعدادات

### للزوار:

- ستظهر أيقونات التواصل الاجتماعي تلقائياً في أسفل الصفحة الرئيسية
- يمكن النقر على أي أيقونة للانتقال إلى الصفحة المقابلة

## المنصات المدعومة

1. **Facebook** - أيقونة زرقاء
2. **Twitter** - أيقونة سماوية
3. **Instagram** - أيقونة وردية
4. **TikTok** - أيقونة سوداء
5. **YouTube** - أيقونة حمراء
6. **WhatsApp** - أيقونة خضراء
7. **LinkedIn** - أيقونة زرقاء داكنة

## ملاحظات تقنية

- يتم تخزين الإعدادات في MongoDB
- يستخدم النظام TRPC للتواصل بين Frontend و Backend
- جميع الروابط اختيارية - يمكن ترك أي حقل فارغاً
- يتم عرض الأيقونات فقط للمنصات التي تم إدخال روابط لها
- النظام يدعم التحديث الفوري دون الحاجة لإعادة تشغيل السيرفر

## التطويرات المستقبلية المقترحة

- إضافة معاينة مباشرة للروابط
- إمكانية رفع شعار المتجر بدلاً من إدخال رابط
- إضافة المزيد من منصات التواصل الاجتماعي
- إحصائيات النقرات على روابط التواصل الاجتماعي
- تخصيص ترتيب ظهور الأيقونات
